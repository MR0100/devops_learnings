# L29/C04/T04 — Meta's Pillars

## Learning Objectives

- Pass Meta behavioral
- Show Meta values

## Meta Values

- Move Fast
- Be Bold
- Focus on Impact
- Be Open
- Build Social Value

## Pillars (Eng)

- Hands-on coder
- Ship quality
- Move fast
- Mentor
- Cross-functional
- Strategy (Staff+)

## Format

- Behavioral round (1 hour)
- Hiring manager round
- Bar raiser (Meta has variation)

## Common Questions

- Tell me about a project you shipped.
- Mistakes / failures.
- Cross-functional disagreement.
- Hire / fire.
- Impact at scale.

## Stories

- Ship: "X feature, 6 weeks, impact Y"
- Failure: "Bad design; learned Z"
- Disagreement: "PM wanted A; I argued B; outcome"
- Impact: "1M users affected by my change"

## Be Bold

Stories where:
- Took risk
- Pushed boundary
- Innovative

## Move Fast

- "Decided in 24 hr; shipped in week"
- "Sometimes break things"
- Learn

## Focus on Impact

- Quantify
- User value
- Business value

## Be Open

- Acknowledge mistakes
- Listen
- Honest feedback

## Best Practices

- Crisp stories
- Ship-focused
- Impact metrics
- Self-aware

## Common Mistakes

- No quantification
- "We" only
- Hide failures
- No initiative

## Quick Refs

```
Values:
- Move fast
- Be bold
- Focus on impact
- Be open
- Build social value

```
