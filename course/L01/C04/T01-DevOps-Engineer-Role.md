# L01/C04/T01 — The DevOps Engineer Role

## Learning Objectives

- Define the day-to-day work of a DevOps Engineer
- Identify what differentiates this role from adjacent roles
- Build a target skill profile

## Typical Day-to-Day

- Building and maintaining CI/CD pipelines
- Writing Infrastructure as Code (Terraform, CloudFormation)
- Managing Kubernetes clusters
- Configuring monitoring and alerting
- Responding to incidents and production issues
- Supporting product teams in deploying their services
- Capacity planning and cost optimization
- Security patching, certificate rotation, day-2 ops

## Required Core Skills

- **Linux**: deep familiarity, comfortable troubleshooting
- **Networking**: VPCs, DNS, load balancers, TLS
- **Cloud provider**: deep on at least one (AWS preferred)
- **Containers**: Docker, Kubernetes
- **IaC**: Terraform (or equivalent)
- **CI/CD**: Jenkins, GitHub Actions, GitLab CI, or ArgoCD
- **Programming**: Bash + Python or Go
- **Observability**: Prometheus, Grafana, ELK or equivalent
- **Security basics**: IAM, secrets, scanning

## Differentiation from Adjacent Roles

| vs. SRE | DevOps engineers tend to be more pipeline/tooling focused; SREs more reliability/SLO focused |
| vs. Platform Engineer | Platform engineers build internal products; DevOps engineers often serve specific teams |
| vs. Cloud Engineer | Cloud engineers focus on cloud architecture (especially AWS-deep); DevOps spans dev tooling |
| vs. SysAdmin | SysAdmins manage existing systems; DevOps automates and ships continuously |

## Common Job Description Phrases & Translation

| Phrase | Reality |
|---|---|
| "End-to-end ownership" | You'll be on-call |
| "Self-starter" | Limited mentorship |
| "Wear many hats" | Limited specialization, broad work |
| "Greenfield environment" | Either exciting or "we have nothing yet" |
| "Modernize legacy systems" | A migration project, possibly painful |
| "Build a platform team" | An IDP investment, exciting but slow |

## Salary & Growth Outlook (2026)

- US median DevOps engineer: $130K–$180K
- Senior: $180K–$280K
- Staff at FAANGM: $350K–$600K+ TC
- Growth path: DevOps → SRE → Staff SRE → Principal Engineer / Platform Architect

## Interview Prep

**Mid**: "What does a DevOps engineer do day-to-day?"

**Senior**: "How is your role different from an SRE at your previous job?"

**Staff**: "If I told you we had no DevOps engineers, what would you build first and why?"
