# L01/C05 — DORA & SPACE Metrics

## Chapter Overview

We introduced DORA in C01/T05. This chapter goes deep: how to measure each metric in your actual environment, the math behind them, the SPACE framework, and how to read these metrics together for a complete picture.

## Learning Outcomes

- Instrument all four DORA metrics in a real CI/CD system
- Read the metrics together to diagnose flow vs reliability vs human cost
- Add SPACE as a counterweight to pure DORA optimization
- Defend metric choices against common criticisms (Goodhart's Law)
