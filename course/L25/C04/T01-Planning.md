# L25/C04/T01 — Planning a Game Day

## Learning Objectives

- Plan game day
- Choose scenario

(Detailed in L19/C07/T03 — referencing.)

## Plan

1. Scenario
2. Hypothesis
3. Scope / blast
4. Roles
5. Schedule
6. Comms plan
7. Rollback

## Scenarios

- DB primary failure
- Region outage
- DDoS
- Bad deploy
- Cache failure
- Network partition

## Scope

- One service
- Cross-team
- Full org

## Roles

- Chaos master
- IC
- Responders
- Observers
- Scribe

## Pre-Brief

1 day before:
- Plan walkthrough
- Materials
- Q&A

## Risk Management

- Stop conditions
- Rollback ready
- Real on-call aware

## Best Practices

- Quarterly minimum
- Cross-functional
- Document plan
- Stop if customer impact
