# L29/C08/T03 — Recruiter Outreach

## Learning Objectives

- Get found
- Respond to outreach

## LinkedIn

- Profile: complete
- Keywords: same as resume
- Headline: clear
- Open to work (sometimes)

## Headline

Strong:
- "Staff Site Reliability Engineer | Kubernetes, AWS, Multi-Region"

Weak:
- "Software Engineer"

## Photo

Professional. Friendly.

## About

3-4 paragraphs:
- Current role
- Strengths
- Looking for

## Experience

Same as resume:
- Impact
- Quantify
- Tech stack

## Skills

Add many:
- LinkedIn algorithm uses
- Endorsements

## Recommendations

Ask former managers / colleagues.

## Activity

- Engage thoughtfully
- Comment on industry
- Share projects

For: visibility.

## Open to Work

```
LinkedIn → Open to Work → Visible to recruiters
```

For: signal interest privately.

## Recruiter Messages

Respond promptly:
- Thank
- Show interest (or not)
- Ask about role
- Don't oversell

## Standard Template

```
Hi [Name],

Thanks for reaching out about [role]. I'd love to learn more. Could we schedule a 15-min call?

A few questions:
1. What's the team focus?
2. What's the level / comp range?
3. Timeline?

Looking forward to chatting.

[You]
```

## Cold Outreach

If you reach out:
- Personalized (mention something specific)
- Brief
- Clear ask

## Active vs Passive

### Active
Apply + LinkedIn outbound.

### Passive
Open to Work; recruiters find.

For: both.

## Best Practices

- Strong profile
- Open to Work (when looking)
- Respond fast
- Polite always

## Common Mistakes

- Profile empty
- No keywords
- Slow respond
- Lie

## Quick Refs

```
Profile: complete + keywords
Headline: clear level + skills
Open to Work: signal
Respond: 24 hr
```
