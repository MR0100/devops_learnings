# L29/C06/T01 — Compensation Components

## Learning Objectives

- Understand comp
- Negotiate effectively

## Components

### Base
Annual salary.

### Bonus
% of base; varies.

### RSU (Restricted Stock Units)
Equity over time (vesting).

### Sign-On
One-time (year 1, year 2).

### Refresher
Annual RSU grants.

## Total Comp

```
Year 1 = Base + Sign-on Y1 + RSU/4 + Bonus
Year 2 = Base + Sign-on Y2 + RSU/4 + Bonus
Year 3 = Base + RSU/4 + Bonus
Year 4 = Base + RSU/4 + Bonus
Year 5 = Base + Refreshers + Bonus
```

(Approximation; varies by company.)

## Vesting

### Standard (Most)
4-year, 25% / year, 1-year cliff.

### Amazon (Back-Loaded)
5% / 15% / 40% / 40%.
Plus sign-on Y1/Y2 to bridge.

### Google / Meta
Even quarterly after 1-yr cliff.

## Sign-On

Bridges back-loaded:
- Amazon: heavy
- Others: moderate

For: don't forget when comparing.

## Refresher

Annual:
- Every year: new RSU grant
- Stacks over time
- Replaces value lost to original vesting

## Negotiation Levers

### Base
Usually band-capped.

### Sign-on
Often most flexible.

### RSU
Sometimes flexible (esp. with competing offer).

### Level
Negotiate up (rare).

## Real Numbers (2024-2025 approx)

### Senior (L5/E5):
- Base: $180-220k
- Bonus: 15-20%
- RSU: $200-400k/4 yrs
- Sign-on: $50-100k
- Total Y1: $300-500k

### Staff (L6/E6):
- Total Y1: $500-700k+

### Sr Staff (L7/E7):
- Total Y1: $700k-1M+

(Varies wildly by company, location.)

## Tools

## Best Practices

- Know total comp
- Don't fixate on base
- Compare offers properly
- Negotiate sign-on / RSU

## Common Mistakes

- Focus only on base
- Underestimate refreshers
- No data
- Accept first offer

## Quick Refs

```
Components: Base + Bonus + RSU + Sign-on + Refresher
Vesting: typically 4 yrs, 1-yr cliff
Negotiate: sign-on, RSU (sometimes base)
```
