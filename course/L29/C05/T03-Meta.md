# L29/C05/T03 — Meta (Move Fast, Impact)

## Learning Objectives

- Prep Meta
- Show impact

## Roles

- E3-E6: SWE
- Production Engineer (PE) = SRE/DevOps
- E5+: senior

## Levels

- E3-E4: Junior-Mid
- E5: Senior
- E6: Staff
- E7: Sr Staff
- E8: Principal

## Process

1. Recruiter
2. Phone screen (coding)
3. Onsite:
   - Coding (2)
   - System design (1, senior+)
   - Behavioral (1-2)

## Production Engineer

For PE / SRE roles:
- Systems focus
- Less algorithm
- More Linux, networking, ops

## Coding

- LeetCode Medium
- Some custom (systems)
- Talk through

## System Design

For senior+:
- Strong design round
- Trade-offs
- Scale

## Behavioral

Move fast + impact:
- Ship-focused stories
- Quantified impact
- Decision speed

## Quantify

For Meta: critical:
- "1M users affected"
- "$500k revenue"
- "10x faster"

## Failure

Acceptable:
- "Broke X; learned Y; now Z"
- Move fast = sometimes break

For: balance.

## Compensation

- Base ~$200k+
- Bonus 10-25%
- RSU heavy
- 1-year cliff

For: front-loaded.

## Best Practices

- Quantified impact stories
- Move-fast examples
- System design ready
- PE: systems focus

## Common Mistakes

- No metrics
- No ship-focus
- All-clean (no learning)
