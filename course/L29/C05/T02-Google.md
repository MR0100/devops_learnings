# L29/C05/T02 — Google (Googleyness, GCA, Domain)

## Learning Objectives

- Prep Google
- Pass hiring committee

## Roles

- SWE
- SRE
- Cloud SE

## Levels

- L3-L4: SWE
- L5: Senior SWE
- L6: Staff SWE
- L7: Senior Staff
- L8: Principal

## Process

1. Recruiter
2. Phone screen (coding)
3. Onsite:
   - Coding (2)
   - System design (1, senior+)
   - Behavioral (1 - Googleyness/Leadership)
   - Domain (varies)

## Hiring Committee

No interviewer veto:
- Packet to HC
- HC decides
- Independent review

## Coding

- LeetCode Medium-Hard
- Algorithm focus
- Optimal solution expected
- Multiple problems per round

For: practice hard.

## System Design

L5+: required.
- 35-45 min
- Drive discussion
- Trade-offs

## Behavioral

Googleyness:
- Ambiguity comfort
- Action
- Collaboration
- Excellence
- Curiosity

Leadership:
- Lead by influence
- Mentor
- Drive

## GCA

General Cognitive Ability:
- Problem-solving
- Reasoning
- Sometimes case-style

For: practice unfamiliar problems.

## SRE

Google invented SRE:
- Deep ops knowledge
- Production excellence
- SLO/SLI/SLA

## Compensation

- Base + bonus + RSU
- Sign-on
- Refresher annual

Generally:
- L5: $400-600k
- L6: $600-900k
- L7: $900k-1.5M

## Best Practices

- Hard algorithms practice
- Google SRE book
- Googleyness stories
- System design for level

## Common Mistakes

- Only Medium LeetCode
- Weak system design
- Skip Googleyness
- No domain depth
