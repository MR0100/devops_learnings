# L20/C09/T02 — ISO 27001

## Learning Objectives

- Know ISO 27001
- Compare to SOC 2

## ISO 27001

International standard:
- Information Security Management System (ISMS)
- Risk-based approach
- 114 controls (Annex A)

## Why

- International recognition
- EU customers
- Government contracts
- Some enterprise demands

## Components

### ISMS
Documented system:
- Scope
- Policies
- Risk methodology
- Controls

### Risk Assessment
Identify, evaluate, treat risks.

### Statement of Applicability
Which Annex A controls apply; why.

### Continuous Improvement
PDCA cycle.

## Annex A Controls

14 categories:
- A.5 Info security policies
- A.6 Org of info security
- A.7 HR security
- A.8 Asset management
- A.9 Access control
- A.10 Cryptography
- A.11 Physical security
- A.12 Operations security
- A.13 Communications security
- A.14 System acquisition, dev, maintenance
- A.15 Supplier relationships
- A.16 Incident management
- A.17 BCP / DR
- A.18 Compliance

## Audit

3-year cert cycle:
- Year 1: full audit
- Years 2-3: surveillance
- Year 4: recertification

Stage 1: docs review.
Stage 2: implementation audit.

## Cert Body

- BSI
- Bureau Veritas
- DNV
- LRQA
- TÜV

## Cost

- First audit: $20-50k
- Annual: $5-15k
- Plus prep / tooling

## Tools

Same as SOC 2:
- Drata, Vanta, Secureframe
- ISO-specific support

## Vs SOC 2

| | ISO 27001 | SOC 2 |
|---|---|---|
| Origin | International | US |
| Cert | yes (3yr) | no (report) |
| Audit | annual | annual |
| Scope | flexible | TSC-defined |
| Controls | 114 (Annex A) | per TSC |
| Recognition | global | mostly US |

For: international: ISO. US-only: SOC 2 enough. Both: best.

## Many Have Both

For global SaaS:
- SOC 2 for US enterprise
- ISO 27001 for EU/global

Drata/Vanta: support both.

## Risk Treatment

Per risk:
- Accept (tolerable)
- Mitigate (control)
- Transfer (insure)
- Avoid (stop activity)

Documented.

## SoA

Statement of Applicability:
- For each Annex A control:
  - Applicable?
  - If not, why?

Drives audit scope.

## ISMS Documentation

Required:
- Scope
- Policy
- Risk methodology
- Risk assessment
- Risk treatment plan
- SoA
- Procedures (some)

## Continuous Improvement

PDCA:
- Plan
- Do
- Check
- Act

Internal audits.
Management reviews.

## Common Controls

Like SOC 2 + more:
- Asset inventory
- Classification
- Crypto policy
- Physical security
- Supplier eval

## Specific to ISO

- Asset inventory (formal)
- Information classification
- Physical security (data center)

## Implementation Time

- Small org: 6-12 months
- Larger: 12-18 months
- Renewal: easier

## Best Practices

- Risk-based mindset
- Documented
- Continuous improvement
- Internal audits
- Management commitment

## Common Mistakes

- Treat as paper exercise
- Skip risk assessment
- No management buy-in
- No improvement

## Compliance Stack

Many orgs:
- SOC 2
- ISO 27001
- HIPAA / PCI / etc. (if relevant)

Drata / Vanta map all.
