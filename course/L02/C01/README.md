# L02/C01 — Linux Fundamentals

## Chapter Overview

Mastering the basics with depth — distributions, filesystem hierarchy, permissions, and the commands you'll use 1000s of times.

## Learning Outcomes

- Pick the right distro for a workload
- Know where to look for any system file
- Set permissions correctly *and securely*
- Use find/xargs/grep at production scale

## Quick Reference Card

```
Distros: Debian/Ubuntu (apt) · RHEL/CentOS/Rocky (dnf) · Alpine (apk) · Arch (pacman)
FHS: /etc · /usr · /var · /opt · /srv · /tmp · /run · /proc · /sys
Perms: rwx for u/g/o · setuid/setgid/sticky · ACL via setfacl · caps via getcap
```
